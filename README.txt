══════════════════════════════════════════════════════
  ميزان POS — تطبيق Windows
══════════════════════════════════════════════════════

📁 الملفات:
  src/index.html     التطبيق الكامل
  main.js            ملف Electron الرئيسي
  preload.js         جسر الاتصال الآمن
  package.json       إعدادات المشروع
  icon.ico           الأيقونة (7 أحجام)
  build.bat          بناء ملف .exe
  run.bat            تشغيل مباشر

══════════════════════════════════════════════════════
🚀 تشغيل مباشر (للاختبار)
══════════════════════════════════════════════════════
1. ثبّت Node.js من: https://nodejs.org (LTS)
2. انقر مرتين على: run.bat

══════════════════════════════════════════════════════
📦 بناء installer للعميل
══════════════════════════════════════════════════════
1. ثبّت Node.js
2. انقر مرتين على: build.bat
3. الملفات في: dist\
   ميزان POS Setup 3.1.0 x64.exe     ← Windows 10/11
   ميزان POS Setup 3.1.0 ia32.exe    ← Windows 7/8/10

══════════════════════════════════════════════════════

══════════════════════════════════════════════════════

  db.json          قاعدة البيانات
  backups\         نسخ احتياطية تلقائية
